## はじめに ― 「人生の木」

終活を始めるにあたり、「自分の想い」を形にするために、人生を一本の木にたとえてみました。木は地中に根を張り、土から栄養をもらって、幹を支えています。幹は少しずつ太くなり、枝が伸び、葉が茂りやがて花や実を付けます。

私達の人生も、自分が生まれた場所や環境（根）、現在の生活や活動（土）があり現在の自分がいるのです（幹）。自分の夢や希望、願いを枝のように広げた先に、自分にとって大切な存在があります（葉）。その大切な存在にしてあげたいこと、残したいもの、さらに自分が思い描くこれからの生活が花や実となります。

これらの「自分の想い」を具体的にエンディングノートに書いてみることで、より明確な「人生の木」が出来上がります。

あなたが描いた「人生の木」を実現するためにご活用いただくのがこのガイドブックです。

## 1. 終活への不安

終活に関する様々な項目を分かりやすく解説します。

終活で不安に感じる主な項目：

- **医療・看護** ― 病気やケガへの備え
- **認知症・介護** ― 認知機能低下への対策
- **終末医療** ― 延命治療などの意思表示
- **葬儀・埋葬・お墓** ― 葬送に関する準備
- **相続・遺言** ― 財産の承継
- **その他** ― 上記以外の備え

## 2. 相続への不安

相続に関して多くの方が抱える不安：

- **相続争い（争族）** ― 遺産分割をめぐるトラブル
- **お一人様** ― 身寄りのない方の相続問題
- **二次相続・老々相続** ― 高齢化に伴う複雑な相続
- **空家になるのが心配** ― 不動産の管理問題

### 相続人関係図

遺言書がない場合には、「相続人関係図」に示す順位で相続人を確認し、相続人同士で遺産分割の話し合いをします。

**法定相続人の順位：**

- **常に相続人：** 配偶者
- **第1順位：** 直系卑属（子・孫）
- **第2順位：** 直系尊属（父母・祖父母）
- **第3順位：** 兄弟姉妹（甥・姪）

### 不動産の手続き

不動産の新規登記、財産の名義変更について：

- 配偶者（妻）→ 子へ
- 相続登記（義務化：令和6年4月1日施行）により、相続により不動産を取得した相続人は、相続の開始を知り、かつ所有権を取得したことを知った日から3年以内に相続登記をすることとなります。これに違反すると10万円以下の過料が科される場合があります。
- 遺産分割協議（話し合い）→ 各相続間の手続きは司法書士等へ

### 相続の対象になるもの・ならないもの

#### 相続の対象になるもの

1. 現金・預貯金
2. 有価証券
3. 土地・家屋などの不動産（山林・田畑など）
4. 貴金属・骨重品・自動車
5. 契約上の地位（賃貸人、賃借人など）
6. 借金・連帯保証人など

※マイナスの財産も含みます。

#### 相続の対象にならないもの

1. **一身専属権** ― 扶養請求権・生活保護受給権・身元保証人としての地位・親権などの権利など
2. **祭祀財産** ― 墓石・墓地・仏壇・仏具・位牌
3. **相続人が受取人になっている生命保険金や死亡退職金** ― ※「みなし相続財産」と呼ばれ相続税の対象となります。

### 相続の各手続きスケジュール

| 時期                                 | 手続き内容                                                                               |
| ------------------------------------ | ---------------------------------------------------------------------------------------- |
| **死亡（7日以内）**                  | 被相続人の死亡（相続開始）、死亡届の提出、通夜・告別式、葬儀、戒名（法名）、遺言書の確認 |
| **相続人の確定**                     | 遺産があるか確認、相続人の確定                                                           |
| **相続放棄・限定承認（3ヶ月以内）**  | 相続放棄または限定承認の手続き                                                           |
| **所得税・準確定申告（4ヶ月以内）**  | 被相続人の所得税の準確定申告                                                             |
| **遺産分割協議**                     | 相続財産の調査・確認、遺産分割協議書の作成                                               |
| **相続税の申告・納付（10ヶ月以内）** | 相続税の申告・納付                                                                       |
| **不動産の名義変更**                 | 不動産の各種登記（※相続登記は3年以内に義務化）                                           |

※「遺言書」がない場合は、相続人全員の話し合いで、遺産分割協議をして、不動産の登記等をはじめ、各手続をしていくことになります。

### ① 生前贈与・生命保険・相続税対策

#### 生前贈与

**概要：** 贈与者が生きているうちに、受贈者に財産を無償で譲ることをいいます。家族以外の他人にも譲ることができます。

**生前贈与が可能な財産：**

- 現金や預貯金
- 土地建物・著作権
- 有価証券・車や船舶
- 家具・宝石・絵画など

※贈与契約書を作成して残すことが大切です。

#### 生前贈与のメリット・デメリット

**メリット：**

1. 贈与税、相続税を軽減する対策ができます。
2. 財産を贈与する時期・相手を自由に選ぶことができます。
3. 贈与者の意思を受贈者に直接伝えることができます。

**デメリット：**

1. 相続が開始される前の一定期間内に贈与された財産は、相続税の課税対象となる場合もあります。
2. 生前贈与された財産が遺留分侵害額請求権の対象になる場合もあります。
3. 贈与契約書が無いと、贈与と認められない可能性が高まります。

※全ての方にあてはまるものではありません。生前贈与に関する課税につきましては税務署又は管轄税務署にご確認ください。

### ① 生命保険

**概要：** 生命保険を活用することで、ご自身の医療費・介護費を確保でき、ご家族にお金を残すこともできます。

#### ＊残された人生に……『リビングニーズ』の選択

生命保険にリビングニーズ特約がついている場合、もし余命6ヶ月以内と診断されたとき、生きているうちに保険金を受けることができます。

- 生きるための治療にお金を使うことができます。
- 残された時間で思い出をつくるためにお金を使う本人が自由に選択をすることができます。
- 上限は3,000万円まで……受け取ったお金には所得税が課されません。

#### ＊生命保険を活用すること

1. 死亡保険金は、「500万円 × 法定相続人の人数」まで非課税枠があります。（非課税枠が認められる保険契約）
   - 契約者（被相続人）→ 被保険者（被相続人）→ 受取人（相続人）
   - ※養子がいる場合は法定相続人の数と異なる場合があります。

2. 死亡保険金は、受取人の固有財産になります。
   - 法律上、受取人自身の財産なので、故人の遺産を相続しないことにはなりません。
   - 原則、遺産分割の対象にはなりません。また、相続放棄をしても受け取れます。
   - 残したいと思うご家族（相続人）に残すことができます。ただし、非課税枠を超える死亡保険金は、みなし相続財産として相続税の課税対象になります。

3. 受取人は、現金を速やかに受け取れます。
   - 死亡保険金は、受取人の請求があれば速やかに受け取れます。葬儀費用の支払い、相続税の納付などの支払いに充てることができます。受取人となった方（相続人）にとっては心強い味方です。

#### 相続税対策

##### 非課税枠の制度概要

| 制度名                                             | 内容                                                                                                                                                                                                    |
| -------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **贈与税の基礎控除**                               | 個人が1年間（1月1日～12月31日）にもらった財産の合計額（複数人から贈与を受けた場合はその合計額）が110万円以内であれば贈与税はかかりません。                                                              |
| **相続時精算課税制度**                             | 60歳以上の贈与者（親・祖父母）が、その年の1月1日において18歳以上の受贈者（子・孫）に生前贈与する際に利用可能な制度です。贈与額累計2,500万円以内であれば贈与税がかからず、相続税の申告の際に精算します。 |
| **住宅資金や増改築費用を贈与する**                 | 家を新築で建てるための資金や、増改築するための費用を贈与する場合には、住宅取得資金贈与の特例が利用可能です。利用条件はありますが、最大1,000万円までの贈与であれば贈与税がかかりません。                 |
| **教育資金を贈与する**                             | 教育資金の一括贈与をした場合、一定の要件を満たせば、受贈者1人につき最大1,500万円まで、贈与税が非課税となります。                                                                                        |
| **結婚費用や子育て資金を贈与する**                 | 結婚・子育て資金の一括贈与をした場合、一定の要件を満たせば、受贈者1人につき最大1,000万円（結婚資金は300万円）まで、贈与税が非課税となります。                                                           |
| **夫婦間で居住用の不動産やその購入資金を贈与する** | 夫婦間で居住用不動産やその取得資金の贈与を受けた場合、婚姻期間が20年以上など一定の要件を満たせば、最大2,000万円まで、贈与税が非課税となります。                                                         |

※2024年3月現在の情報です。税法は変更される場合がありますので、適用にあたっては最新の情報をご確認下さい。

### ② 遺言書作成

**自分の想い：** 自分の死後、相続でもめてほしくない。残された家族の手を煩わせたくない → 自分の想いを遺言書に託します。

**こんな方は、遺言書の作成が必要です：**

- 子供がいない
- 再婚で前の配偶者との間に子供がいる
- 内縁関係である
- 相続人がいない
- 相続人となる人が認知症である
- 会社経営者・個人事業主

#### 遺言書の種類

| 種類                             | 説明                                                                                                                |
| -------------------------------- | ------------------------------------------------------------------------------------------------------------------- |
| **自筆証書遺言（自分で保管）**   | 財産目録以外は自筆で記載します。書いた日付、署名、押印が必要です。死後に家庭裁判所の検認が必要です。                |
| **自筆証書遺言（法務局で保管）** | 自分で保管する自筆証書遺言と同じ形式で記載します。作成した遺言書を法務局で保管してもらう制度があります。            |
| **公正証書遺言**                 | 公証人に遺言の趣旨を伝え、公証人に遺言書を作成してもらいます。2名の証人が必要です。遺言書は、公証役場で保管します。 |

それぞれにメリット・デメリットがあります。どのような遺言書にするか、遺言書の具体的な書き方を行政書士がご支援いたします。

### ③ 終末期医療

医療行為を受けるか否かの決定権者は、ご本人のみですが、終末期医療の現場において、意思表示を行うことが困難な場合もあります。その際に、事前に終末期医療に対する意思を明確にしておくことが重要です。

#### 終末期医療における意思表示

| 項目     | 内容                                                                                           |
| -------- | ---------------------------------------------------------------------------------------------- |
| **目的** | 終末期における、人間としての尊厳を保ったまま死を迎えるため（延命治療の拒否・中止）のものです。 |
| **方法** | 尊厳死宣言書（リヴィング・ウイル）を家族と話し合った上で作成し、医療機関に提示します。         |

**尊厳死宣言書の作成方法：**

1. 公証役場で「尊厳死宣言公正証書」を作成し公証役場で保管してもらいます。
2. 日本尊厳死協会に加盟し尊厳死宣言書を作成し保管してもらいます。
3. 医療機関指定の終末期における事前指定書等を作成・提出します。

**効果：** 現在、尊厳死に関する法律の整備はいまだ検討中のため、尊厳死が合法化されているわけではありません。そのため、尊厳死宣言書が医療機関に対する法的拘束力を有するとまでは断言できませんが、厚生労働省発の「人生の最終段階の医療・ケアの決定プロセスに関するガイドライン」において、本人の意思決定の有無につき尊厳死宣言書を考慮する旨の考え方が示されており、また家族の精神的な助けになることが見込まれます。

**撤回：** いつでも尊厳死宣言を破棄・撤回することが可能で、そのための特別な手続は要しません。もっとも、撤回する場合はトラブル防止の観点から書面で残しておくことを推奨いたします。

### ④ 墓じまい・永代供養・散骨

**関連法令：** 墓地、埋葬等に関する法律（墓埋法）

#### 墓じまい

| 項目     | 内容                                                                                                                                                                                                                                                               |
| -------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **目的** | お墓の管理が困難となり、お墓に納められている遺骨を取り出し、墓石を処分する等して墓所を整理し、墓地区画を寺院や霊園に返却することです。                                                                                                                             |
| **方法** | 遺骨を新たに他の墓地や納骨堂に納める場合は、各市区町村役場での改葬手続が必要となります。手元供養の場合は、「改葬」にはあたらないため法律上特段の手続は不要です。ただ、その他に寺院や霊園が定める規則等もあるため、寺院や霊園の管理者に必ず確認の上行ってください。 |

#### 永代供養

| 項目     | 内容                                                                                                                                                                                                                                                   |
| -------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **目的** | 祭祀承継者がおらず、お墓参りをしてくれる人がいない、またはお墓参りに行けないため、代わりに寺院や霊園に管理や供養をしてもらうための埋葬方法です。個人墓と比較すると経済的負担は軽くなますが、永代供養料等は各寺院や霊園の管理者にお問い合わせください。 |
| **方法** | 寺院や霊園の管理者に永代供養料を支払ってお願いします。                                                                                                                                                                                                 |

#### 散骨

| 項目     | 内容                                                                                                                                                           |
| -------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **目的** | 遺骨を埋葬するのではなく、個人の希望に基づき海や山に遺灰を撒いて死者を弔うことです。                                                                           |
| **方法** | 特段の手続は必要はないですが、公衆衛生上の問題や散骨場所の所有者や近隣住民等との関係から十分注意を要します。自治体によっては、散骨に関する規制を設けています。 |

## 6. 2nd「信頼できる人」にお願いする

### 概要

2nd「信頼できる人」にお願いするステップでは、以下の5つの項目を扱います：

1. 家族信託®・親なき後問題・終活空家信託®・コラム・ペット信託®
2. 死後事務委任・遺言執行・遺贈寄付
3. 任意後見
4. 身元保証
5. 養子縁組

---

### ① 家族信託（民事信託）

**行政書士支援可能**

**関連法令：** 信託法（平成19年9月30日改正）

**目的：** 信託銀行や信託会社に資産を預けるのではなく、信頼できる方に財産を託し、費用を抑えた形で財産管理と資産承継を行うものです。認知症対策と相続対策双方を行うことが可能です。

**方法：**

1. **信託契約** ― 財産を預ける者（委託者）と財産を預かる者（受託者）との間で信託契約を締結し、同契約によって利益を受ける者（受益者）のために、契約内容に従って受託者が財産管理や資産承継を行います。
2. **遺言信託** ― 上記1）で定める内容を遺言の中で設定し、遺言者の死後の財産管理や資産承継を行います。上記1）と異なり、委託者の財産を包括的に信託財産に含めることが可能となります。
3. **自己信託** ― 自己信託とは、委託者が自ら受託者となり、受益者のために自己の財産を管理（信託宣言）・処分します。

※上記1から3の契約書、遺言書又は自己信託設定書を作成する際は、家族信託の専門家（行政書士、司法書士、税理士等）に相談のうえ公証役場で作成することを推奨いたします。

**撤回：** 信託が終了するのは、信託法で定める終了事由の発生又は委託者、受益者の同意を得て解除するほか、信託契約等で別途終了事由や変更事由を定めておけば、その事由の発生により終了又は変更（一部撤回）させることが可能となります。

※注意：民事信託も万能ではないため他の制度との併用が必要な場合があります。

#### 民事信託のイメージ図

- **委託者・遺言者（例：親）** ― 財産を預ける人
- **受託者（例：子供）** ― 財産を預かる人 → 管理・処分・運用
- **受益者（例：親）** ― 預けた財産の利益を享受する人

※信託監督人や受益者代理人の定めをおいて行政書士など専門家に監視・監督させることも可能です。

**受託者に移転するもの：**

- 不動産は所有権移転登記及び信託登記により名義を変更
- 金銭・有価証券は受託者の信託専用口座等を開設の上移管

**信託財産の例：** 自宅などの所有不動産、金銭等

#### 親なき後問題（福祉型信託）

障がいのある子ども又は何らかの事故が原因で後遺症が残り自分で財産管理ができない子どもをもつ親が病気・死亡により、子どもの生活支援ができなくなったとき……信頼できる人に子どもの生活を託すための信託契約を福祉型信託と呼ばれています。

**福祉型信託のイメージ図：**

- **父（委託者兼当初受益者）** → 信託契約 → **信頼できる親族又は第三者（受託者）**
- 受託者は託された信託財産を管理
- 父の死後、財産を管理して、子どもの生活費をサポート
- **子ども（障がいあり）** ＝ 2次受益者（父の死後、子どもが受益者へ）
- **受益者代理人（任意）** ― 当該受益者が判断能力の低下 → 代理人として受益者の利益を守る

#### 終活空家信託®

**背景：** 現在自宅に一人暮らし。そろそろ安心できる施設へ入居しようかしら……。子供達は遠方に住んでおり、自宅に住む予定もなく空家になるリスクが高い。

**問題点：** 高齢の母が認知症になり判断能力が低下・喪失すると、自宅の賃貸・売却・解体などをすることが出来なくなります。その場合は、法定後見制度を利用することになります。判断能力がしっかりしているうちに、終活空家信託制度をご検討ください！

**仕組み：**

- **委託者 兼 受益者：** 高齢の母（固有財産）
- **受託者：** 長男（信託財産）
- 高齢の母と長男の間で信託契約を締結
- **信託契約の内容：** 生活費の支給、空家の管理・賃貸・売却・解体

**メリット：**

- 高齢の母が認知症になっても、長男が母の自宅の賃貸・売却・改築・解体をすることが可能になります。
- 不動産の賃料・売却代金は信託口座で管理し、母の生活費に充てることが可能です。
- 管理費用、改修費用や解体費用は信託口座から捻出しますので、長男の負担がなくなります！

#### ペット信託

**概要：** ペット信託とは、信託法に基づいて、飼い主が大切なペットを飼育できなくなったとき（施設入所・お亡くなりになったとき）に備えて、飼い主が元気なうちに信頼できる人にペットの飼育と飼育費を託す契約のことです。

**ペット信託のイメージ図：**

- **飼い主（委託者兼受益者）** → 猫の飼育と飼育費の信託契約 → **飼育費を管理する者（受託者）**
- 受託者 → 飼育費を送る → **猫の飼育先（飼い主が飼育できなくなった場合、猫を飼育する者）**
- **信託監督人（任意）** ― 監督

### ② 死後事務委任・遺言執行・遺贈寄付

#### 死後事務委任

自分が亡くなった後の様々な手続きを、信頼できる人とあらかじめ契約を結び、お願いすることができます。相続人がいない、親族が離れて暮らしているため、手続きを行うことが困難である……そんな方は信頼できる方にお願いしましょう。

**主な死後の事務：**

- 死亡届（任意後見人、任意後見受任者になっている場合）
- 葬儀、納骨
- 水道、電気、ガス、電話等の公共料金の手続き
- 健康保険、介護保険、住民税、固定資産税等の役所での手続き
- 年金手続き

#### 遺言執行

遺言書を作成するときに、遺言書の内容を実行してくれる人（遺言執行者）を指定しておくと、スムーズに手続きを行うことができます。

遺言執行者の指定がない場合、家庭裁判所に申立をして選任してもらうこともできます。

#### 遺贈と遺贈寄付

**遺贈とは：** 遺言書により無償で財産を譲ることです。

**財産を引き継ぐ人の違い：**

|                | 相続       | 遺贈                                                 |
| -------------- | ---------- | ---------------------------------------------------- |
| **引き継ぐ人** | 法定相続人 | 誰でもOK（法定相続人以外の第三者、法定相続人、法人） |

**不動産の手続きの違い（相続 vs 遺贈）：**

| 項目                     | 相続                                               | 遺贈                                                       |
| ------------------------ | -------------------------------------------------- | ---------------------------------------------------------- |
| **不動産の登記**         | 相続人が一人で申請できます。                       | 受遺者と相続人全員または受遺者と遺言執行者で申請します。   |
| **第三者への権利の主張** | 登記がなくても主張できます。（法定相続割合の範囲） | 登記していないと主張できません。                           |
| **農地**                 | 許可は不要です。（届出は必要）                     | 農地法による農業委員会又は、都道府県知事の許可が必要です。 |
| **借地権・借家権**       | 貸主の承諾は不要です。                             | 貸主の承諾が必要です。                                     |

※相続と遺贈、遺贈の種類や者により、課される税金や税率が異なるので、注意が必要です。

#### 遺贈寄付

社会貢献活動に役立ててもらう目的で、自分の死後、遺産を特定の団体に寄付することです。国や地方公共団体、一定の公益法人等への遺贈寄付は、相続税を減額する効果があります。

寄付する団体にあらかじめ何を寄付するのかを伝えておくことが大切です。

**遺言による寄付：** 遺言書に、財産を寄付することを記載しておきます。お世話になった施設や病院、自治体や慈善団体等に寄付することを遺言書に書いておきます。

**相続財産による寄付：** 相続財産を受け取った人が寄付をすることです。寄付をしたいという内容の事柄がエンディングノートに書かれていたり、生前、本人から聞いていた相続人が、本人に代わって寄付することです。

※遺贈寄付は、課税関係が複雑ですので、注意が必要です。

### ③ 任意後見

#### 任意後見制度

判断能力があるうちに、認知症になったり、障害を持った場合に備えて、あらかじめ信頼できる方（任意後見受任者）に代わりにしてもらいたいことを公正証書による契約で決めておく制度です。

**任意後見の3つのステップ：**

**ステップ1：本人に十分な判断能力がある時**

- 本人が選んだ任意後見受任者と任意後見契約を締結
- 公証人が任意後見契約の公正証書を作成
- 登記

**ステップ2：本人の判断能力が低下した時**

- 任意後見監督人選任の申立（本人、任意後見受任者等から）
- 家庭裁判所が任意後見監督人を選任

**ステップ3：任意後見の開始**

- 任意後見人が本人を保護・支援
- 任意後見監督人が任意後見人を監督

身元保証人をお願いできる身寄りがいない……。信頼できる人が身近にいることは大切なことなんですね。

### ④ リバースモーゲージ、リースバック

#### リバースモーゲージ

**目的：** 居住用不動産（自宅）はあるが、年金などの現金収入が少ない場合、自宅不動産を活用し、住み慣れた自宅に住み続けながら生活資金や介護費用を捻出することが目的です。

**内容：** 居住用不動産に担保を設定し、一定額の金銭を定期的に借り入れ、本人が亡くなったとき、居住用不動産を売却して、借入金を返済します。通常の住宅ローンは、一度にまとまったお金を借りて、少しずつ返しますがリバースモーゲージは、少しずつ借りて、最後にまとめて返済します。また、一度に借り入れし、借入期間中に利息のみ支払い、亡後に自宅を売却して元本を返済するタイプもあります。

**取扱い：** 社会福祉協議会が制度を運営している、公的な制度があります。住宅金融支援機構のリ・バース60や一部の民間の金融機関でも扱いがあります。公的制度以外で生活資金を融資することはできなく、建替え、リフォーム、借り換え等住宅費用に限定しているところが多いです。

#### リースバック

**内容：** 不動産業者との間で、自宅を借り戻す条件で売却する契約を結びます。その後、売却した自宅に住み続けながら家賃（リース）を不動産業者に支払います。自宅が賃貸になることで、維持費が軽減されます。また売却代金を生活資金、介護費用、趣味や賃料に充てていきます。借り戻す条件は、リースバックを行う不動産業者によって異なります。

**取扱い：** 民間不動産業者です。

- **遺族厚生年金** ―
  - 第1順位：配偶者（夫の場合は、妻の死亡時に55歳以上）
  - 第2順位：子（満18歳になって初めて迎える3月31日までの方。20歳未満で障害年金の障害等級1級又は2級の状態にある方）
  - 第3順位：父母（死亡時55歳以上）
  - 第4順位：孫（子と同じ年齢制限あり）

**手続き：** 年金事務所に連絡をして、死亡届、未支給年金の請求、遺族年金について確認しましょう。

- 亡くなった方が加入していた年金（国民年金、厚生年金（旧共済年金を含む））や、遺族の状況によって支給される遺族年金が違ってきます。
- 手続きをしてから、支給開始までには、相当な期間を要します。早めの手続きをお勧めします。

### ⑥ 日常生活自立支援事業

| 項目     | 内容                                                                                                                                                                           |
| -------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **目的** | 認知症や精神障がい等により、日常生活を送るうえで支障がある方に、福祉サービスの利用に関する相談や助言、利用のために必要な手続き、または費用の支払い等の援助を目的にしています。 |

**対象者：** 次のいずれにも該当する方。審査があります。

1. 認知症高齢者、知的障がい・精神障がいのある方で、判断能力が不十分なため、日常生活を送るうえで支障がある方
2. 本人だけでは、生活するうえで必要なサービスを利用するための情報の入手、理解、判断、意思表示を適切に行うことが困難な方
3. 援助サービスの契約の内容について判断できる能力を有していると認められる方

**援助内容：**

1. 日常的な生活支援サービス → 福祉サービスの利用のお手伝い
2. 金銭的管理サービス
   - a. 公共料金や税金、社会保険料、医療費等を支払う手続き
   - b. 年金や福祉手当の受領に必要な手続き
   - c. 日常生活費の管理のお手伝いや、それに伴う預金の預入、払戻手続き
3. 財産保全サービス → 年金証書、定期預金通帳等、普段使わない書類等を金融機関の貸金庫でお預かり

これらのサービスを比較的お手頃な料金で利用することができます。

**申請窓口：** 市区町村の社会福祉協議会・地域包括支援センターです。

## 8. 人生の木を描く終活会議

「終活ガイドブック」を通じて、ご本人様の人生の木を描くために、信頼できるご家族又は信頼する誰かと終活会議で「話し合い」「共有」することが大切です。

### 人生の木のイメージ

中心に **「ご本人の想い」** があり、そこから枝葉のように大切な人々への想いが広がります：

- **配偶者** ― 家と生活費を残したい
- **長男** ― 家の管理をお願いしたい
- **長女** ― 妻の生活を支えてほしい
- **母** ― 医療費・介護費を残したい
- **孫** ― 教育費を残したい
- **弟** ― 母の老後を支えてほしい
- **友人** ― お世話になっている現に何か残したい
- **ペット** ― 飼育できなくなったときに、飼育と飼育費を残したい。大切なペットをお願いできる人も大切ですね。
